Welkom

Op deze website vindt je de online Arduino editor van Stichting HET LAB Rotterdam. Hier kan je verschillende lessen vinden die je kan doen via de online editor.


De volgende lessen zijn op dit moment ontwikkeld:

1. Hello Blink

In deze les zal je de basis leren van een Arduino, hierbij wordt gebruik gemaakt van de sketch Hello World. Het is de bedoeling om een lampje te laten knipperen. Deze opdracht is de equivalent van de Hello World opdrachten die je als eerste maakt in andere programmeertalen. Hier leer je het fenomeen Digital Output.

Ga naar de les →

2. Stoplicht Bouwen

In deze les zal je leren hoe je meerdere lampjes kan aansturen met een knopje. Hierbij komt Digital Input aanbod.

Ga naar de les →

3. Sensor input met een Light Dependent Resistor(LDR)

In deze les leer je omgaan met een simpele sensor, een Light Dependent Resistor(LDR), dit is een lichtsensor. Hierbij ga je gebruik maken van Analog Input en daarbij de verschillende lampjes aansturen die je in voorgaande lessen heb gebouwd. Het is de bedoeling dat met minder licht op de sensor er meerdere lampjes gaan branden.

Ga naar de les →

4. Geluid maken met een Piezo Speaker

In deze les leer je Analog Output door middel van PWM(Pulse Width Modulation), dit is een techniek dat een analoog signaal simuleert over een digitaal signaal.
